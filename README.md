# Asimov-infinity
Asimov Infinity is a scifi-themed ecommerce store built using React,Express , MirageJS, HTML and CSS.

## Features Present 
- Authentication
    - Login
    - Logout
    - Signup
- Product Listing
- Filter Products by
    - Category
    - Discount
    - Price
    - Rating
- Cart Management
- Wishlist Management
- Search by 
  - Product Name
  - Category
- Address Management
- Single Product Page
- Loading & Alerts
- User Profile Page
- Checkout
- Order Summary
- Order History
- Apply Coupons
- Payment Integration
- Pagination on Product Listing Page
